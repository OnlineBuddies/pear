Online Buddies' PEAR Channel Server

Registering the channel: 
pear channel-discover onlinebuddies.github.com/pear
Listing available packages: 
pear remote-list -c OnlineBuddies
Installing a package: 
pear install OnlineBuddies/package_name
Installing a specific version/stability: 
pear install OnlineBuddies/package_name-1.0.0
pear install OnlineBuddies/package_name-beta
Receiving updates via a feed: 
http://onlinebuddies.github.com/pear/feed.xml
