Online Buddies' PEAR Channel Server

Registering the channel: 
pear channel-discover onlinebuddies.github.com/pear
Listing available packages: 
pear remote-list -c OnlineBuddies
Installing a package: 
pear install OnlineBuddies/package_name
Installing a specific version/stability: 
pear install OnlineBuddies/package_name-1.0.0
pear install OnlineBuddies/package_name-beta
Receiving updates via a feed: 
http://onlinebuddies.github.com/pear/feed.xml

PEAR_PackageFileManager_Gitrepoonly - A Git repository-only list plugin generator for both PEAR_PackageFileManager, and PEAR_PackageFileManager2 classes.

It will only include files that are part of the git index-- that is, files in the current branch plus files staged for commit. It simply uses `git ls-files` to get a list of files to package. If you just want files excluded that are in .gitignore or .git/info/exclude, see https://github.com/armen/PEAR_PackageFileManager_Git

Install commandpear install OnlineBuddies/PEAR_PackageFileManager_Gitrepoonly
